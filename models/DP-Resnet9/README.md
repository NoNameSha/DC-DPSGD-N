# Resnet9

## Dependencies
- pytorch = 1.8
- cuda 10.2
- torchvision
- matplotlib

## Training:

Uncompress the folder in any location and run the following command

`python test_geo.py --train PATH_TO_TRAINIG_DATA`

The training data folder must have 2 folders called set_test and set_train
